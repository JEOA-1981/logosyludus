# logos&ludus

Página de proyecto de empresa científica. 
Logos & Ludus se propone como una empresa que fomenta la generación y difusión del conocimiento de manera colaborativa a través de la ludificación; en otras palabras, los mismos usuarios comparten e intercambian sus conocimientos, construyendo comunidades mediante juegos.
El flujo de trabajo de la empresa contempla las siguientes fases:
a) Obtención de datos mediante web scraping o uso de APIs, para identificar los intereses de usuarios
b) Preprocesamiento y análisis de datos para analizar patrones y tendencias de usuarios
c) Preparación de modelo de Aprendizaje Automático, para aprovechar los patrones identificados
d) Diseño de aplicación móvil que se sustente en elementos lúdicos
e) Ensayo, despliegue y marketing de aplicación
